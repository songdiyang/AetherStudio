## 重要声明

**1. 赞助性质**

本项目的赞助为**纯自愿的无偿捐赠**，不包含任何形式的商品或服务交换。赞助者不会因此获得任何额外功能、优先支持、专属权益或其他福利。

**2. 年龄限制**

**未成年人（未满18周岁）禁止进行赞助**。如果您是未成年人，请在监护人陪同下使用本项目，且不得进行任何形式的财务捐赠。

**3. 理性赞助**

成年人在赞助前，请务必：

- 确认您已年满18周岁，具备完全民事行为能力；
- 评估自身经济状况，确保赞助金额在您的可承受范围内；
- 充分理解赞助为无偿捐赠性质，不会产生任何回报或权益；
- 确认您了解本项目为开源软件，赞助不构成就业、投资或商业合作关系。

**4. 退款说明**

由于赞助属于自愿捐赠性质，**原则上不支持退款**。如有特殊情况，请通过 GitHub Issues 联系维护者协商处理。

**5. 资金用途**

所有赞助资金将用于项目的持续开发、服务器维护、域名续费及相关运营成本。维护者保留对资金用途的最终解释权。

**6. 免责条款**

- 本项目按「现状」提供，不对任何直接或间接损失承担责任；
- 赞助行为完全基于您的自愿，维护者不对赞助者的任何决策或行为负责；
- 如您不同意以上条款，请勿进行赞助。
